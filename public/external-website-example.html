üîê Authentication Modes

1. Guest Mode

Users can browse and shop without any authentication. Perfect for quick interactions.

<script src="aladdyn-genie-external.js"></script> <script> AladdynGenie.init({ authMode: 'guest', shopDomain: 'your-store.myshopify.com', apiBaseUrl: 'https://your-api.com/api' }); </script>

2. Email Authentication

Users enter their email to verify against your Shopify customer database.

<script> AladdynGenie.init({ authMode: 'email', shopDomain: 'your-store.myshopify.com', apiBaseUrl: 'https://your-api.com/api' }); </script>

3. Hybrid Mode (Recommended)

Users can choose between guest mode or full authentication. Best user experience!

<script> AladdynGenie.init({ authMode: 'hybrid', enableGuestMode: true, shopDomain: 'your-store.myshopify.com', apiBaseUrl: 'https://your-api.com/api' }); </script>

4. Social Login

Quick authentication with Google, Facebook, or Apple Sign-In.

<script> AladdynGenie.init({ authMode: 'social', socialProviders: ['google', 'facebook', 'apple'], shopDomain: 'your-store.myshopify.com', apiBaseUrl: 'https://your-api.com/api' }); </script>

5. Store Account Authentication

Connect directly to customer's Shopify store account for full access to personal cart, orders, and checkout.

<script> AladdynGenie.init({ authMode: 'hybrid', // Includes store account option enableGuestMode: true, shopDomain: 'your-store.myshopify.com', apiBaseUrl: 'https://your-api.com/api', // Store account OAuth will be handled automatically }); </script>

6. Custom Authentication

Integrate with your existing website authentication system.

<script> // Get user data from your auth system const userData = { name: 'John Doe', email: 'john@example.com', id: 'user123' }; AladdynGenie.init({ authMode: 'custom', customAuthData: userData, shopDomain: 'your-store.myshopify.com', apiBaseUrl: 'https://your-api.com/api' }); </script>

üöÄ Advanced Configuration

<script> AladdynGenie.init({ // Authentication authMode: 'hybrid', // 'guest', 'email', 'social', 'custom', 'hybrid' enableGuestMode: true, // Allow guest browsing socialProviders: ['google', 'facebook', 'apple'], // Store Configuration shopDomain: 'your-store.myshopify.com', apiBaseUrl: 'https://your-api.com/api', // Custom Authentication customAuthData: null, // User data for custom auth // Callbacks onAuthSuccess: function(authData) { console.log('User authenticated:', authData); // Handle successful authentication }, onAuthError: function(error) { console.error('Auth error:', error); // Handle authentication errors } }); </script>
